A block to displaying stars, indicating the highest rating of satisfaction or quality, with a total up to ten stars.

== Description ==

Gutenberg star rating blocks are visual elements in the WordPress Gutenberg editor, allowing users to easily add and customize star ratings to their content, enhancing user engagement and feedback.

## FEATURES ##
 - Choosing a star rating up to 10 stars
 - Easily assign the star rating scale with the increment of 0.5 star
 - Allow user to customize the title with difference font and size
 - Allow user to change the gap between the star and the label
 - Easy customize the sizing of the star and label


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rating-stars-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
